# One Piece Manga Tracker

Pagina online: https://tpicchio.github.io/op_manga_tracker/

Piccolo tracker personale per tenere traccia dei volumi di One Piece gia comprati, cosi da averli sempre disponibili e non comprare duplicati.

## Funzioni principali

- Lista volumi sempre visibile anche senza login
- Ricerca volume per testo
- Filtro Nascondi comprati
- Barra progresso acquisti
- Indicazione visiva per arco narrativo

## Permessi

- Utente non loggato: sola lettura
- Utente loggato: può segnare come comprato, aggiungere il prossimo volume ed eliminare l'ultimo volume

## Stack

- Frontend statico: HTML, CSS, JavaScript
- Backend/Auth/Database: Supabase
- Hosting: GitHub Pages

## Struttura file

- index.html: struttura pagina e inclusione asset
- styles.css: stili UI
- app.js: logica applicativa, auth e chiamate Supabase

## Obiettivo del progetto

Tenere un unico riferimento semplice e veloce per controllare i manga gia acquistati prima di comprarne altri.
